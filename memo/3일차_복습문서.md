## 1. EDA 개요

데이터의 구조, 성격, 내용을 파악하는 작업으로, 아래 세 가지 라이브러리를 주로 활용한다.

- numpy: 벡터, 행렬 연산에 편의성을 제공하는 라이브러리
- pandas: 데이터 프레임 형태로 데이터 분석에 편의성을 제공하는 라이브러리
- visualization (matplotlib, seaborn): 데이터 시각화 라이브러리

## 2. numpy

### 개념 요약

- ndarray 기초: 1차원, 2차원 배열 생성
- 같은 모양의 배열끼리 동일 위치 성분 간 사칙연산
- axis=0 은 행 방향, axis=1 은 열 방향
- 주로 쓰는 데이터 타입: int, float, bool

### 배열 생성

| 함수 | 설명 |
|------|------|
| `np.array()` | 리스트를 배열로 변환 |
| `np.zeros()` | 0으로 채운 배열 생성 |
| `np.ones()` | 1로 채운 배열 생성 |
| `np.arange()` | 연속된 숫자로 배열 생성 |
| `np.random.randn()` | 정규분포 난수 배열 생성 |
| `np.random.randint()` | 정수 난수 배열 생성 |
| `np.random.permutation()` | 배열 순서 무작위 섞기 |

### 파일 입출력

| 함수 | 설명 |
|------|------|
| `np.genfromtxt()` | 파일에서 데이터 로드 |
| `np.savetxt()` | 배열을 파일로 저장 |

### 데이터 타입

| 함수 | 설명 |
|------|------|
| `arr.dtype` | 데이터 타입 확인 |
| `arr.astype()` | 데이터 타입 변환 |

### 연산 함수

| 함수 | 설명 |
|------|------|
| `np.add()` | 덧셈 |
| `np.subtract()` | 뺄셈 |
| `np.multiply()` | 곱셈 |
| `np.divide()` | 나눗셈 |
| `np.maximum()` | 두 배열 중 큰 값 반환 |
| `np.minimum()` | 두 배열 중 작은 값 반환 |

### 통계 함수

| 함수 | 설명 |
|------|------|
| `arr.sum()` | 합계 |
| `arr.mean()` | 평균 |
| `arr.std()` | 표준편차 |
| `arr.var()` | 분산 |
| `arr.min()` / `arr.max()` | 최소 / 최대 |
| `arr.argmin()` / `arr.argmax()` | 최소 / 최대값의 인덱스 |
| `arr.cumsum()` | 누적합 |
| `arr.cumprod()` | 누적곱 |

### 정렬 및 기타

| 함수 | 설명 |
|------|------|
| `np.sort()` | 오름차순 정렬 (내림차순은 `[::-1]`) |
| `np.unique()` | 중복 제거 후 고유값 반환 |

## 3. pandas

### 개념 요약

- Series: 1차원 자료구조
- DataFrame: 2차원 자료구조 (엑셀과 유사, 컬럼명 기반 인덱싱 가능)
- loc: 컬럼명/인덱스명 기준 인덱싱
- iloc: 자리수(숫자) 기준 인덱싱
- axis=0 은 행 방향, axis=1 은 열 방향

### 자료구조 생성

| 함수 | 설명 |
|------|------|
| `pd.Series()` | 시리즈 생성 |
| `pd.DataFrame()` | 데이터프레임 생성 |
| `pd.date_range()` | 날짜 범위 인덱스 생성 |
| `pd.to_datetime()` | 문자열을 날짜 타입으로 변환 |
| `pd.read_csv()` | csv 파일 로드 |

### 정보 확인

| 함수 | 설명 |
|------|------|
| `df.index` | 인덱스 확인 |
| `df.columns` | 컬럼 확인 |
| `df.values` | 값 확인 |
| `df.dtype` | 타입 확인 |
| `df.shape` | 형태 확인 |
| `df.count()` | 개수 확인 |
| `df.head()` | 앞 N개 행 확인 |
| `df.tail()` | 뒤 N개 행 확인 |
| `df.describe()` | 평균, 분산, 최소/최대 등 기본 통계량 한번에 출력 |

### 인덱싱

| 함수 | 설명 |
|------|------|
| `df.loc[]` | 컬럼명/인덱스명 기준 인덱싱 |
| `df.iloc[]` | 자리수(숫자) 기준 인덱싱 |

### 조작

| 함수 | 설명 |
|------|------|
| `df.drop()` | 행 또는 열 삭제 (axis=0 행, axis=1 열) |
| `df.dropna()` | nan 값 포함 행 삭제 (how='any' / how='all') |
| `df.fillna()` | nan 값을 지정한 값으로 대체 |
| `df.isnull()` | nan 여부 true/false로 확인 |
| `df.reindex()` | 인덱스/컬럼 재정렬 |
| `del df['열이름']` | 열 삭제 |

### 분석

| 함수 | 설명 |
|------|------|
| `df.sum()` | 합계 (axis 방향 지정) |
| `df.mean()` | 평균 (skipna 옵션으로 nan 행 스킵 여부 설정) |
| `df.min()` / `df.max()` | 최소 / 최대 |
| `df.corr()` | 변수 간 상관계수 산출 |
| `df.pivot_table()` | 피벗 테이블 생성 |

### 정렬 및 검색

| 함수 | 설명 |
|------|------|
| `df.sort_index()` | 인덱스 기준 정렬 (ascending=False 로 내림차순) |
| `df.sort_values()` | 값 기준 정렬 (ascending=False 로 내림차순) |
| `df.isin()` | 특정 값 포함 여부 true/false |

### 시각화

| 함수 | 설명 |
|------|------|
| `df.plot()` | 데이터프레임 기반 그래프 출력 |

## 4. 핵심 포인트

- axis 방향: 0은 행 방향(열 기준 연산), 1은 열 방향(행 기준 연산)
- loc vs iloc: loc은 이름 기준, iloc은 숫자 위치 기준
- boolean 인덱싱은 실무에서 매우 자주 쓰이는 패턴
- 수치 데이터만으로는 설득이 어렵고, 시각화를 통해 인사이트를 전달해야 효과적
- nan 처리 (dropna, fillna, isnull) 는 실무 데이터 전처리에서 필수
